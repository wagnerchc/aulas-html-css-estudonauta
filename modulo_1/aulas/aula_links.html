Tag <a href="" alt="" target="">

  • target="_blank"

    • O site abrirá em outra aba do navegador por conta do atributo target="_blank"

    Clique aqui para visitar o site do Google

  • target="_self"

    • O site abrirá na mesma aba do navegador por conta do atributo target="_self"

    Clique aqui para visitar o site do Google

  • target="_top"

    • O site vai desfazer todos os frames e abrir o destino no navegador completo por conta do atributo target="_top"

    Clique aqui para visitar o site do Google

  • target="_parent"

    • O site abrirá semelhante ao atibuito "_top", desfazendo todos os frames e abrirá em referência à janela mãe por conta do atributo target="_parent"

    Clique aqui para visitar o site do Google

  • target="nome-do-frame"

    • A técnica de frames está quase em desuso, mas a ideia seria indicar o nome da janela a abrir usando o atributo target="nome-do-frame"

    Clique aqui para visitar o site do Google

Tag <a href="" alt="" target="" hreflang="">

O atributo hreflang="" indica qual é o idioma principal do site para onde o link está desviando o fluxo de navegação. Isso sinalizará ao navegador e a softwares de tradução como lidar caso o visitante opte por traduzir automaticamente os conteúdos.

  • Site do Google.com com a <hreflang="en"> indicando o site em inglês Google.com
  • Site do Google.com com a <hreflang="pt-br"> indicando o site para traduzir para português Google.com.br

Tag <a href="" alt="" target="" hreflang="" rel="">

O atributo rel="" indicar qual é a natureza do destino

  • next - indica que o link é para a próxima parte do documento atual
  • prev - indica que o link é para a parte anterior do documento atual
  • author - indica que é um link para o site do autor do artigo atual
  • external - indica que é um link para outro site que não faz parte do site atual
  • nofollow - indica que é um link para um site não endossado, como um link pago
